Dragonfly: 02 ImageMagick plugin
================================

```
$ ./01encode.rb
$ open out.jpg
# => show jpeg image

$ ./02thumb.rb
$ open out.jpg
# => show thumbnail image

$ ./03grayscale.rb
$ open out.jpg
# => show grayscale image

$ ./04encode_low_quality.rb
$ open out.jpg
# => show low quality image
```
